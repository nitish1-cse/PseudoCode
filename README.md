## 💡 Core Concepts Covered

### 🔹 Modular Programming
- Functions (return values)
- Procedures (void methods)
- Code reusability and clarity

### 🔹 Control Flow
- If, else-if ladder, nested if
- Loops (for, while)
- Real examples like factorial, reverse number

### 🔹 Bitwise Operations
- XOR (^), AND (&)
- Right Shift (>>)
- 1’s and 2’s complement

### 🔹 Arrays & Matrices
- 1D array operations (sum, search)
- 2D arrays (matrix addition, subtraction)
- Row-wise traversal

### 🔹 Problem Solving Logic
- Armstrong number
- Palindrome
- Even/Odd
- Coordinate quadrant
